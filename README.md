# IQView

High-performance Static RF Spectrogram Viewer.

## Installation

For users:
```bash
pip install iqview-0.1.0-py3-none-any.whl
```

For developers (Setup):
```bash
# Clone the repository and run:
pip install -e .
```

## Usage
Once installed, use the `iqview` command:
```bash
iqview -f path/to/file.iq -t complex64 -r 2e6
```
